# mk-piclock v1.7.6 Release Notes

## Clean OLED network diagnostics

- Removed unsupported parentheses from the OLED Wi-Fi signal line.
- Signal now appears as `75% -62 dBm` instead of `75% (-62 dBm)`.
- This removes the fallback question mark previously shown after `dBm`.
- README now says to simply hold down the touch sensor to open network diagnostics, without specifying the internal hold time.
- Music and Story Mode gesture handling is unchanged.
